Equiprose

Equipoise [n]: Equality in distribution, as of weight, relationship, or emotional forces; equilibrium.

Equiprose is a static website and blog generator written in Node.js

Features

  • Static website and blog generator
  • Markdown support
  • Native emojis support
  • Write your templates with Jade and Stylus
  • Website preview
  • Ease of use and customization
  • Simple, yet flexible and powerful

Install

$ [sudo] npm install -g equiprose

Basic usage

  1. $ equiprose new ~/website: creates a website in ~/website

  2. $ equiprose serve ~/website: compile and serve the website at localhost:8888
