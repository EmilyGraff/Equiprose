Equiprose

Equipoise [n]: Equality in distribution, as of weight, relationship, or emotional forces; equilibrium.

Equiprose is a static website and blog generator written in Node.js.

Features

  • Static website + blog generator
  • Shipped to you as a simple binary script
  • Markdown support
  • Native emojis support
  • Website preview
  • Templating with Jade and Stylus
  • Multiple and widely configurable websites

Install

$ [sudo] npm install -g equiprose

Basic usage

$ equiprose new ~/website

$ equiprose serve ~/website

Boom.
